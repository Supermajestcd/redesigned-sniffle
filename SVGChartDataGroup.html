Implements IItemRendererParent, IContentView, IChartDataGroup

Language Version : ActionScript 3.0
Product Version : FlexJS 0.0
Runtime Versions : Flash Player 10.2, AIR 2.6

The SVGChartDataGroup serves as the drawing canvas for SVG itemRenderers. Rather than having individual itemRenderer objects in the display list, this class provides a canvas where the itemRenderers can draw directly using the flex.core.graphics package.
